# liri-node-app

-- uses terminal to use an API call to return facts when using a handful of commands:

"concert-this"
Sends an API call to Bands in Town and returns:
Artist Name
Venue
Location
Date

"spotify-this-song"
Sends API call to Spotify and returns:
Artist
Song Name
Spotify Link
Album

"movie-this"
Sends API call to OMDB and returns:
Title
Year
IMDB Rating
Rotten Tomatoes
Country
Language
Plot
Actors
